## 🔍 Project Overview
This project analyzes sales data to understand customer behavior, product performance, and sales trends.

## 🛠 Tools Used
- Python (Pandas, Matplotlib)
- SQL (MySQL)
- Tableau Public

## 📈 Key Insights
- Sales increased in recent months indicating seasonal trends  
- Technology category generated highest revenue  
- Repeat customers contribute significantly to overall sales
